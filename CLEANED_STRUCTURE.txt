WHAT IS THIS PROJECT?
---------------------
ANSE (Agent Nervous System Engine) is a Python framework that gives AI agents 
direct access to hardware tools (cameras, microphones, speakers) while maintaining
security, privacy, and determinism.

CORE IDEA:
- AI agent connects via WebSocket
- Calls hardware tools (capture_frame, record_audio, say, etc)
- All data stays local, fully audited, rate-limited
- Async-safe with proper thread pooling
- Event-sourced for replay & debugging

GETTING STARTED
---------------
1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   pytest tests/ -v

3. Start engine:
   python -c "from anse.engine_core import EngineCore; import asyncio; asyncio.run(EngineCore().run())"

4. Connect agent via WebSocket:
   See examples/scripted_agent.py or examples/llm_agent_adapter.py
