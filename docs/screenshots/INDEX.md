# ANSE Dashboard Screenshots

Add your screenshot image files here (PNG, JPG, etc.)

## How to Add Screenshots

1. **Save screenshot files** in this directory (`docs/screenshots/`)
   - Example: `sensor-panel.png`, `full-dashboard.png`, etc.

2. **Add entry below** with image name and description

3. **Update** `../../SCREENSHOTS.md` to reference the new images

## Example Entry

Once you add a screenshot like `sensor-panel.png`, add this line:

```markdown
| sensor-panel.png | Distance sensor reading live updates | ✅ |
```

Then reference it in `SCREENSHOTS.md`:

```markdown
![Sensor Panel](docs/screenshots/sensor-panel.png)
```

---

## Screenshot Naming Convention

Use clear, descriptive names:
- ✅ `sensor-panel.png`, `actuator-panel.png`
- ❌ `screenshot1.png`, `img.png`
